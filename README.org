** Twitch Commands

*** ctof

~F = num * (9/5) + 32~

#+BEGIN_EXAMPLE text
>  !ctof
=> 0*C = 32*F

>  !ctof -17.78
=> -17.78*C = -0.00*F
#+END_EXAMPLE

*** ftoc

~C = (num - 32) * (5/9)~

#+BEGIN_EXAMPLE text
>  !ftoc
=> 0*F = -17.78*C

>  !ftoc 32
=> 32*F = 0.00*C
#+END_EXAMPLE
